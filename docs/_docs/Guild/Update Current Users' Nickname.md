# `updateNick`

```php
$client->guild->updateNick($parameters);
```

## Description

Updates the bots nickname in a server

## Parameters


Name | Type | Required | Default
--- | --- | --- | ---
guild.id | snowflake | false | *null*
nick | string | false | *null*

## Response

Returns the nick
